Object-Oriented Design & Programming

Project Repository: OODPproject

Project Overview

This project is a Java-based application focused on implementing a Battle System and Inventory Management using Object-Oriented Programming principles.

Core Systems

  • Battle System: Handles combat logic and turn-based mechanics.
  • Inventory System: Manages items, equipment, and player resources.
  • Entity Management: Defines characters, enemies, and NPCs through inheritance.
  • Save System: Implements data persistence via the Saveable interface.

Project Structure

The source code is organized into packages including entity for game objects and exception for robust error handling.
